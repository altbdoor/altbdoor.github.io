Simple Java Compiler
A small script that simplifies the process of Java compilation with console. The script runs java.exe and javac.exe from their respective locations, so there is no need to change environment variables. Besides that, the script also strips file types and replace them with suitable file types when you compile, run or edit the file. Low-level editor is available with GNU nano text editor, or the Windows built-in command line text editor, edit.exe.
